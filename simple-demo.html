Quick Setup - Install XAMPP (Recommended)

Step 1: Download XAMPP

XAMPP includes PHP, MySQL, and Apache web server in one package.

Download XAMPP

Step 2: Install XAMPP

Run the installer and install XAMPP (usually to C:\xampp)

‚úÖ Select: Apache, MySQL, PHP, phpMyAdmin

Step 3: Start Services

Open XAMPP Control Panel and start:

‚úÖ Apache (web server)

‚úÖ MySQL (database)

Both should show green "Running" status.

Step 4: Copy Project Files

Copy your project folder to XAMPP's web directory:

From: C:\Users\mzsp7\Music\Projects\food menu system

To: C:\xampp\htdocs\food-menu-system

Step 5: Set Up Database

1. Open phpMyAdmin

2. Create new database: food_menu_system

3. Import the file: database/schema.sql

Step 6: Access Your Application

Open your browser and go to:

http://localhost/food-menu-system

Login with: admin / admin123

Alternative: Command Line Setup

If you prefer command line:

1. Install PHP from windows.php.net

2. Add PHP to your system PATH

3. Install MySQL separately

4. Run: php -S localhost:8000 in your project directory
